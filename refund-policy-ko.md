## 2. 포인트의 성격 및 정의

### 2.1 포인트의 법적 성격
- 웹빵 포인트(이하 "빵")는 웹빵 서비스 내에서 디지털 콘텐츠(방탈출 게임 등)를 이용하기 위한 선불전자지급수단입니다.
- 빵은 현금이 아니며, 법정화폐로의 교환을 보장하지 않습니다.

### 2.2 포인트의 구분
1. **충전 포인트(P-Points)**: 사용자가 현금으로 직접 구매한 포인트
2. **적립 포인트(G-Points)**: 콘텐츠 제작으로 획득한 수익 포인트

### 2.3 포인트의 제한 사항
- **양도 불가**: 빵은 타인에게 양도, 이전, 증여, 상속할 수 없습니다.
- **현금 전환 불가**: 충전 포인트는 환불 절차를 제외하고 현금으로 전환할 수 없습니다.
- **유효기간**: 충전일로부터 1년간 유효하며, 이용 기간 경과 시 자동 소멸됩니다.
- **환불 가능기간**: 결제시점으로부터 1년 이내에만 환불 신청이 가능합니다.

## 3. 환불 가능 조건

### 3.1 기본 환불 요건
다음의 **모든 조건**을 충족하는 경우에만 환불이 가능합니다:

1. **시간 요건**: 포인트 충전 후 7일 이내
2. **사용 요건**: 충전한 포인트를 전혀 사용하지 않은 경우
3. **본인 요건**: 본인이 직접 충전한 포인트에 한함
4. **유효기간 요건**: 충전일로부터 1년 이내

### 3.2 전자상거래법상 청약철회권
「전자상거래 등에서의 소비자보호에 관한 법률」 제17조에 따라, 위 조건을 충족하는 경우 별도의 위약금 없이 청약을 철회할 수 있습니다.

### 3.3 환불 가능 금액
- 충전한 포인트 **전액**에 대해서만 환불이 가능합니다.
- 부분 환불은 불가능합니다.

## 4. 환불 불가 사유

### 4.1 법적 환불 제한 사유
다음의 경우 「전자상거래법」 제17조 제2항에 따라 청약철회가 제한됩니다:

1. **디지털 콘텐츠 이용 개시**
   - 방탈출 게임을 1회라도 플레이한 경우
   - 유료 콘텐츠를 열람하거나 다운로드한 경우
   - 포인트를 일부라도 사용한 경우

2. **시간 경과**
   - 충전 후 7일이 경과한 경우
   - 유효기간(충전일로부터 1년)이 만료된 경우

3. **재판매 불가**
   - 포인트 사용으로 인해 그 가치가 현저히 감소한 경우

### 4.2 정책상 환불 제한 사유
다음의 경우에도 환불이 불가능합니다:

- 프로모션, 이벤트, 보상으로 무료 지급받은 포인트
- 회원탈퇴 후 신청하는 환불
- 부정한 방법으로 획득한 포인트
- 타인으로부터 양도받았다고 주장하는 포인트
- 시스템 오류가 아닌 사용자 실수로 인한 포인트 사용

### 4.3 서비스 하자 시 예외
다음의 경우 위 제한 사유에도 불구하고 환불이 가능합니다:

- 콘텐츠가 광고 내용과 현저히 다른 경우
- 서비스에 중대한 하자가 있어 이용이 불가능한 경우
- 회사의 귀책사유로 정상적인 서비스 제공이 불가능한 경우

### 5.2 필요 정보
환불 신청 시 다음 정보를 제공해야 합니다:

- 사용자 이름 및 이메일
- 충전 일시 및 금액
- 환불 사유
- 환불 받을 계좌 정보 (은행명, 계좌번호, 예금주)
- 본인 확인 자료 (필요시)

### 5.3 환불 처리 절차

#### 1단계: 신청 접수 (즉시)
- 환불 신청 접수 확인 메시지 전송

#### 2단계: 검토 및 승인 (영업일 기준 1~2일)
- 환불 가능 조건 충족 여부 확인
- 포인트 사용 내역 확인
- 부정 이용 여부 검토

#### 3단계: 환불 처리 (영업일 기준 3~5일)
- 승인 시: 환불 처리 및 완료 안내
- 거부 시: 거부 사유 안내

#### 4단계: 최종 환불 (결제사별 상이)
- 카드 결제: 카드사 승인 취소 (2~7일 소요)
- 계좌 이체: 지정 계좌로 입금 (1~3일 소요)
- 간편결제: 각 결제사 정책에 따름 (3~7일 소요)

### 5.4 환불 방법 우선순위

**원칙**: 결제한 수단과 동일한 방법으로 환불합니다.

1. **신용카드**: 카드사 승인 취소
2. **간편결제**(카카오페이 등): 간편결제 잔액 환급
3. **계좌이체**: 환불 계좌로 송금

**예외**: 다음의 경우 계좌 이체로 환불됩니다:
- 원 결제수단으로 환불이 불가능한 경우
- 카드 해지, 계좌 폐쇄 등의 사유가 있는 경우
- 결제사의 정책상 제한이 있는 경우

### 5.5 환불 수수료
- **수수료 없음**: 회사 귀책 사유 및 정상적인 청약철회는 수수료가 없습니다.
- 단, 금융기관의 송금 수수료는 이용자가 부담할 수 있습니다.

## 6. 포인트 사용처

### 6.1 포인트 사용 용도
빵은 다음의 용도로만 사용할 수 있습니다:

1. **유료 방탈출 게임 플레이**
   - 제작자가 설정한 가격만큼 포인트 차감
   - 1회 구매 = 무제한 플레이 가능

2. **프리미엄 콘텐츠 이용** (향후 제공 예정)
   - 특별 테마팩
   - 제작자 후원 기능

### 6.2 포인트 충전 후 사용 경로

```
[포인트 충전]
    ↓
[내 빵 잔액에 적립]
    ↓
[유료 방탈출 게임 선택]
    ↓
[게임 구매 (빵 차감)]
    ↓
[게임 플레이 시작] ← ⚠️ 이 시점부터 환불 불가
    ↓
[무제한 플레이 가능]
```

**중요**: 게임 플레이를 시작하는 순간 해당 포인트는 환불이 불가능합니다.

### 6.3 포인트 차감 우선순위
여러 종류의 포인트를 보유한 경우 다음 순서로 차감됩니다:

1. 유효기간이 임박한 포인트 우선
2. 무료 적립 포인트 우선
3. 충전 포인트 (오래된 순서)

## 7. 유효기간 및 소멸

### 7.1 포인트 유효기간
- **충전 포인트**: 충전일로부터 **1년**
- **이벤트 포인트**: 지급 시 개별 고지

### 7.2 유효기간 연장
- 포인트 사용 또는 추가 충전 시에도 기존 포인트의 유효기간은 연장되지 않습니다.
- 각 포인트는 충전일 기준으로 독립적으로 관리됩니다.

### 7.3 소멸 안내
- 유효기간 만료 30일 전: 알림 발송 (앱 푸시, 이메일)
- 유효기간 만료 7일 전: 최종 알림 발송
- 유효기간 만료 시: 자동 소멸 (복구 불가)

### 7.4 소멸된 포인트
- 소멸된 포인트는 어떠한 경우에도 복구되지 않습니다.
- 소멸 후 환불 신청은 불가능합니다.
